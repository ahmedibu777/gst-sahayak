Run in 5 minutes

  1. Unzip gst-sahayak-complete-codebase.zip
  2. cd gst-sahayak
  3. pip install -r requirements.txt
  4. Copy .env.example → .env, add Grok API key from x.ai
  5. streamlit run main.py

Opens at localhost:8501

terminal 
cd gst-sahayak
pip install -r requirements.txt
cp .env.example .env
# Add GROK_API_KEY=sk-...
streamlit run main.py

🐳 Docker

docker build -t gst-sahayak . && docker run -p 8501:8501 --env-file .env gst-sahayak

☁️ Streamlit Cloud

Push to GitHub → connect at share.streamlit.io → add GROK_API_KEY in Secrets.
